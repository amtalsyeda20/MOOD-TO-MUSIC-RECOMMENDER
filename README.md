# MOOD-TO-MUSIC-RECOMMENDER
SentiBeat AI is an AI-powered emotion detection and music recommendation system developed using Python, Streamlit, OpenCV, and Deep Learning. The project analyzes human facial expressions either through live webcam detection or uploaded images and predicts the user’s emotion in real time.
Based on the detected mood (Happy, Sad, Angry, Neutral, Surprise, etc.), the system automatically recommends and plays suitable music from the web to enhance the user experience.

The emotion recognition model was trained on 1000+ facial expression images using Convolutional Neural Networks (CNN). The application provides an interactive and user-friendly interface built with Streamlit and runs through the Anaconda environment.

Features
Real-time face emotion detection using webcam
Emotion recognition from uploaded images
AI-based mood classification
Automatic music recommendation according to emotion
Streamlit interactive web interface
Deep Learning model trained on 1000+ images
Technologies Used
Python
Streamlit
OpenCV
TensorFlow / Keras
NumPy
PIL (Python Imaging Library)
Deep Learning (CNN)
How It Works
Capture face through webcam or upload an image
Detect facial expressions using AI model
Predict the emotion/mood
Recommend and play matching music automatically
